# Remote Mower [![Build Status](https://travis-ci.org/tarbadev/remote-mower.svg?branch=master)](https://travis-ci.org/tarbadev/remote-mower)

## Start App
### Start React server
`yarn start`
### Start Electron application
`yarn electron`

## Running Tests
### Unit Tests
`yarn test src`
### E2E Tests
`yarn test e2e`
